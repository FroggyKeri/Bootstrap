Plant Automata

a Project by Til Naumann & Kerstin Heitzmann

Das Projekt

Das Ziel bestand darin, einen kompakten "Roboter" zu entwickeln, der Daten mithilfe spezifischer Sensoren erfassen kann. Als Beispiel wird die Feuchtigkeit des Bodens erfasst, woraufhin dem angeschlossenen Pumpensystem signalisiert wird, die Pflanze zu bewässern. Die Implementierung des Steuerungsprogramms erfolgte auf einem Raspberry Pi 4 unter Verwendung der Programmiersprache C.

Daten

Die Datenerfassung erfolgte über einen Zeitraum von mehreren Tagen und wurde in einer Datei gespeichert. Anschließend wurde diese Datei umstrukturiert und in das JSON-Format konvertiert. Die so erstellte JSON-Datei wurde dann mithilfe von Chart.js und JavaScript, wie in der obigen Darstellung dargestellt, visualisiert und in ein Diagramm integriert.

Website

Die Webseite wurde unter Verwendung von HTML, SCSS und JavaScript entwickelt. Dabei wurde das Open-Source-Framework Bootstrap 5 für die Gestaltung und Strukturierung der Seite eingesetzt. Zusätzlich wurde die Open-Source-JavaScript-Bibliothek Chart.js verwendet, um die Daten visuell darzustellen.
